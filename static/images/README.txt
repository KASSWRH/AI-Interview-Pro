# ملفات الصور

هذا المجلد مخصص للصور المستخدمة في المشروع. يجب إضافة الملفات التالية:

1. `logo.png` - شعار التطبيق
2. `screenshot-home.png` - لقطة شاشة للصفحة الرئيسية
3. `screenshot-analysis.png` - لقطة شاشة لصفحة تحليل السيرة الذاتية
4. `screenshot-voice.png` - لقطة شاشة لصفحة المقابلات الصوتية
5. `screenshot-saved.png` - لقطة شاشة لصفحة الأسئلة المحفوظة

يمكنك التقاط لقطات شاشة للتطبيق باستخدام أداة التقاط الشاشة في جهازك أو أي أداة أخرى، ثم وضعها في هذا المجلد.
